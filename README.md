# Chinmay Singh - Portfolio

Professional portfolio website showcasing backend-heavy full-stack development expertise.

## Tech Stack

- **Framework:** Next.js 14 (App Router, Static Export)
- **Language:** TypeScript (Strict Mode)
- **Styling:** Vanilla CSS + CSS Modules
- **Animations:** Framer Motion
- **Deployment:** Vercel

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build

```bash
npm run build
```

Generates static export in `/out` directory.

### Bundle Analysis

```bash
npm run analyze
```

Analyzes bundle size and composition.

## Performance Budgets

- **JavaScript Bundle:** <150KB gzipped
- **Total Page Load:** <500KB
- **Images:** <200KB per asset

## Project Structure

```
portfolio/
├── public/          # Static assets (resume.pdf, images)
├── src/
│   ├── app/        # Next.js App Router
│   ├── components/ # React components
│   ├── lib/        # Utilities (GitHub API, analytics)
│   ├── data/       # Static content (JSON)
│   ├── styles/     # CSS variables and global styles
│   └── types/      # TypeScript definitions
```
